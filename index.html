Latest Project

Reddit User Pics

Reddit User Photo Search - a website built using React and Tailwind that allows users to search for and view photos uploaded by any Reddit user. The website retrieves the last 100 posts made by the user, and displays any photos found in a grid. Clicking on a photo opens a modal window to view it in full size. This project showcases my proficiency in React, Tailwind, and API integration.

Games

Wordle clone

A clone of the popular game, Wordle, using Javascript. The main objective of the project was to learn about working with APIs. The game is designed to randomly generate a five-letter word that the user has to guess. The project also utilizes an API to retrieve the definition of the word, which is displayed to the user after they successfully guess the word. The key components that I developed include the game logic, user interface, and API integration.
